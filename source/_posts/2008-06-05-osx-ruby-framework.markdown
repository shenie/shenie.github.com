<p>I think new version of ruby is meant to be installed with the following prefix, using 1.8.7 as an example.</p>

<p>/System/Library/Frameworks/Ruby.framework/Versions/1.8.7/usr</p>

<p>That way you can just /System/Library/Frameworks/Ruby.framework/Versions/Current to the version you want.</p>
